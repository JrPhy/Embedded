STM32F429 Air Monitor (PMS5003)
這是一個基於 STM32F429I-Discovery 實作的空氣品質監測專案。透過 USART1 與 PMS5003 顆粒物感測器通訊，利用 DMA 與 空閒中斷 (IDLE Line Detection) 接收數據，並將解析後的 PM2.5 與 PM10 數值即時顯示於開發板內建的 LCD 螢幕上 。  

## 一、功能特點：
1. 高效能數據接收：使用 UART DMA 循環模式搭配空閒中斷，確保數據接收的即時性且不佔用 CPU 資源 。  
2. 精準數據解析：實作 PMS5003 通訊協定解析，包含起始碼校驗、校驗和 (Checksum) 比對與大端模式數據轉換 。  
3. 即時顯示：整合 STM32F429 內建 LCD，即時更新環境中的 PM2.5 與 PM10 濃度 (單位: ug/m<sup>2</sup>)

## 二、狀態指示：每次成功解析一包數據時，板載 LED (GPIOG Pin 13) 會閃爍一次 。  

## 三、硬體配置
1. 開發板：STM32F429I-DISCO (STM32F429ZIT6) 。  
2. 感測器：PMS5003 (UART 輸出，鮑率 9600) 。  

## 四、引腳連接：
1. USART1 RX (PA10)：連接感測器 TX 引腳 。  
2. USART1 TX (PA9)：連接感測器 RX 引腳 (如需指令發送) 。  
3. LCD：使用板載 LTDC 控制器配置之螢幕 。  

## 五、數據解析邏輯
1. 程式碼會檢查 PMS5003 封包的前 32 個位元組：
驗證起始碼 0x42 0x4D 。  計算前 30 位元組的總和並與最後兩位元組的 Checksum 進行比對 。  

2. 解析環境空氣品質數據 (Atmospheric Environment)：\
pm25_env: PM2.5 濃度 與 pm100_env: PM10 濃度 。  

## 六、使用說明
環境需求：需安裝 STM32CubeIDE 或 Keil MDK。\
燒錄程式：編譯並將韌體燒錄至開發板。\
接線：將 PMS5003 感測器依照上述引腳與開發板連接，並提供 5V 電源給感測器。\
監測：開機後螢幕會顯示 "Air Monitor v1.0"，隨後即會出現即時數據 。
